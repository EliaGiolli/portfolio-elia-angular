# 🚀 Personal Portfolio - Angular | Elia Giolli
Welcome to the repository of my new **personal portfolio**. This project is a modern, high-performance web application designed to showcase my journey and expertise as a **Full-Stack Developer**.

`⚠️ Work in Progress: Core features are now implemented. Currently refining accessibility and testing suites. `

## ✨ Project Vision
The goal of this portfolio is to merge a minimalist, "premium" design aesthetic with a robust technical architecture. It serves as a living demonstration of my **proficiency with Angular 19** and modern web standards.

## 🏗️ Architecture & Features
Built with a focus on maintainability and the "Zero-Footprint" personalization strategy:

- Advanced Layout System: Implements nested routing and standalone components to handle diverse UI structures seamlessly.

- Atomic Component Design: A dedicated library of reusable components including custom Buttons, Cards, and an SVG Icon system.

- Reactive Filtering: Project showcase powered by Angular Signals, allowing real-time filtering by technology and stack.

- Safe Form Handling: Contact system built with Reactive Forms and validated through Zod schemas for end-to-end type safety.

- ptimized SVG System: A custom-icon renderer using CSS masks for dynamic coloring without external library overhead.
