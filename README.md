> **⚠️ Work in Progress**: This project is currently under active development. Expect frequent updates as I refine the UI and add new project showcases.

## ✨ Project Vision
The goal of this portfolio is to merge a minimalist, "premium" design aesthetic with a robust technical architecture. It serves as a living demonstration of my proficiency with **Angular 19** and modern web standards.

## 🏗️ Architecture & Features
Built with a focus on maintainability and the "Zero-Footprint" personalization strategy:

* **Advanced Layout System:** Implements nested routing to handle diverse UI structures (Homepage vs. Project Details) seamlessly.
* **Atomic Component Design:** A dedicated library of reusable components including custom Buttons, Cards, and an SVG Icon system.
* **Multi-Slot Content Projection:** Utilizing `<ng-content>` to create highly flexible and accessible semantic cards.
* **Modern State Management:** Leveraging Angular **Signals** for reactive, fine-grained DOM updates.
* **Optimized SVG System:** A custom-built icon renderer that uses CSS masks for dynamic coloring without external library overhead.
